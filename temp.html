Поведенческие реакции участника проекта ОСТРОВ

Взгляд тестировщика: участник размещает мечты, помогает другим, исполняет чужие и свои мечты. Ниже — максимальный набор сценариев и статус реализации.

✅ Реализовано ❌ Не реализовано / частично

ДействиеСтатус
Открыть лендинг (вход/регистрация)✅
Войти по телефону и паролю✅
Зарегистрироваться (имя, фамилия, телефон, город, пароль)✅
Выйти (футер / десктоп)✅
Восстановить сессию из localStorage при обновлении страницы✅

2. Профиль и настройки

ДействиеСтатус
Увидеть в хедере своё имя и аватар (или инициалы)✅
Клик по своей аватарке (режим «Мои мечты») → открыть настройки профиля✅
Изменить пароль (текущий + новый)✅
Загрузить/сменить аватар✅
Указать Telegram, VK, телефон для витрины («Хочу помочь»)✅
Сохранить настройки и вернуться назад✅

Редактировать мечту (текст, дедлайн) из модалки✅
Удалить мечту из модалки редактирования✅
Изменить статус мечты (Запланировано / В работе / Выполнено)✅
Открыть/закрыть мечту для витрины (глазок: публичная / личная)✅
Развернуть мечту и увидеть шаги (аккордеон)✅
Добавить шаг к мечте✅
Отметить шаг выполненным (галочка)✅
Редактировать название/дедлайн шага на лету✅
Удалить шаг (в список «Удалённые»), восстановить из удалённых✅
Свайп между карточками мечт на мобиле✅
Финцель: таблица по месяцам (план/факт), модалка при галочке шага✅

4. Бадди

ДействиеСтатус
Увидеть в хедере слот бадди (аватар + имя) или кнопку «Добавить бадди»✅
Клик по слоту бадди → переключиться на «Мечты бадди»✅
Клик «Я» (в режиме бадди) → вернуться к своим мечтам✅
Открыть модалку «Добавить бадди» → список пользователей без бадди✅
Отправить приглашение в бадди («Пригласить»)✅
Отменить исходящее приглашение («Ждём ответа» → отмена)✅
Увидеть входящий запрос в бадди (в слоте: «Запрос от …», Принять / Отклонить)✅
Принять или отклонить входящий запрос в бадди✅
Просматривать мечты бадди (только чтение или с правом редактирования — buddy_trust)✅
Редактировать мечты бадди, шаги, статус, публичность (если trust)✅

5. Витрина мечт (чужие мечты — помочь другим)

ДействиеСтатус
Открыть витрину (клик «Помог» в хедере или «Витрина мечт» в меню)✅
Увидеть фильтры: Витрина мечт (N) · Новые · Помогаю · Помог · В процессе · Избранные✅
Клик по «Витрина мечт (N)» → показать все мечты✅
Переключать фильтры (Новые, Помогаю, Помог, В процессе, Избранные)✅
Авто-переключение на непустой фильтр при пустых «Новые»✅
Поставить лайк (сердечко) → мечта в «Избранные», владельцу уведомление✅
Снять лайк → мечта уходит из избранного✅
Увидеть счётчик лайков (0/1 без цифры, 2+ с цифрой)✅
Нажать «Хочу помочь!» → модалка с текстом и контактами (Telegram, VK, телефон)✅
Скопировать сообщение, перейти в мессенджер по ссылке✅
Нажать «Готово!» (в карточке мечты из «Помогаю») → запрос на завершение владельцу✅
Нажать «Вернуть» → перестать помогать по этой мечте✅
Увидеть в «Помогаю» мечты, по которым я помощник✅
Увидеть в «Помог» мечты, которые я помог исполнить✅
Вернуться к своим мечтам («← К своим мечтам» или клик по «N сбылось»)✅

6. Владелец мечты: уведомления и «В процессе»

ДействиеСтатус
Увидеть индикатор на колокольчике при новых уведомлениях✅
Открыть уведомления (клик по колокольчику) → список в контенте✅
Уведомление «X завершил(и) твою мечту» + кнопки Принять / На доработку✅
Уведомление «Вашу мечту добавили в избранное»✅
Нажать «Принять» → модалка: в завершённые или в личные мечты (одноразовая/повторяемая)✅
Подтвердить «В завершённые» или «В личные» → мечта обновлена, помощник получил «Помог»✅
Нажать «На доработку» → помощник остаётся в «Помогаю», может снова нажать «Готово!»✅
Открыть фильтр «В процессе» на витрине → мои мечты, по которым помогают✅
В карточке «В процессе»: кнопки «Принять» и «Отказаться»✅
«Отказаться» → отказаться от всех помощников по этой мечте✅

10. Что не реализовано или частично

Действие / сценарийКомментарий
Раздел «Расписание» как отдельное имя (сейчас «Шаги»)❌ В roadmap — переименование
Сводное расписание: все шаги по датам, «что сделать сегодня»❌ Сейчас шаги по мечтам + фильтры, не единый календарный вид
Несколько бадди (сейчас один)❌ Планово в roadmap
Пометить уведомления «прочитанными» (снять точку)❌ Точка по pending_completion; отдельно «прочитано» для избранного не хранится
Отказаться от одного помощника (сейчас «Отказаться» = от всех)❌ МВП — от всех; в help_flow_spec отмечено «позже доработать»
Единый стиль футера на десктопе и мобиле (явно синхронизирован)❌ В structure зафиксировано как цель
Итого: большинство поведенческих сценариев участника (размещение мечт, помощь другим, исполнение своих и чужих мечт, бадди, витрина, лайки, уведомления, шаги) реализованы. Не покрыты: переименование «Шаги» в «Расписание», сводное расписание по датам, несколько бадди, точечный отказ от одного помощника, явная синхронизация футера.
