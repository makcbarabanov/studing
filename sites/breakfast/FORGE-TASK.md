## Продукт (MVP)

Оффлайн-проект «Завтрак, исполняющий желания» → лид-магнит соцсети Остров.

1. Короткое описание + примеры на странице.
2. Кнопка **«Записать свою мечту»** → модалка / overlay с **Conversational UI**.
3. ИИ «Света» (тёплая, эмпатичная) — **2–3 вопроса**, лёгкая декомпозиция мечты.
4. Итог: данные в **боевую БД** → пользователь с мечтой и шагами в **ЛК** `web-app`.

**Не в MVP:** полный парсер логов чата, Bloom Telegram, сложная аналитика.
