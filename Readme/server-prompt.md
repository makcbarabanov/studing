# Промпт на сервере — чтобы не путать с локальной машиной

Чтобы сразу видеть, что ты в SSH на сервере (island), а не на локальном компе (Makc), на **сервере** в `~/.bashrc` добавь в конец:

```bash
# Явно показываем, что это сервер (остров)
if [ -n "$SSH_CONNECTION" ]; then
  export PS1='\[\033[01;31m\][SERVER island]\[\033[00m\] \u@\h:\w\$ '
fi
```

Или вариант с зелёным «остров»:

```bash
if [ -n "$SSH_CONNECTION" ]; then
  export PS1='\[\033[01;32m\][ОСТРОВ]\[\033[00m\] \u@\h:\w\$ '
fi
```

После правки выполни `source ~/.bashrc` или зайди по SSH заново.

- **Локально:** `makc@Makc:~$` (как обычно).
- **На сервере:** `[SERVER island] makc@island:~$` или `[ОСТРОВ] makc@island:~$` — сразу видно, что это прод.
